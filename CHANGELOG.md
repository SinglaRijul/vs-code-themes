# Changelog

All notable changes to Earthy Themes will be documented here.

## [1.0.0] - 2026-04-02

### Added
- **Terra** theme — warm earthy palette with burnt oranges, clay, and sandstone tones
- **Moss & Stone** theme — cool earthy palette with sage greens, fern, and lichen tones
- Full workbench color coverage (editor, sidebar, activity bar, terminal, status bar, tabs, panels)
- Syntax token colors for all major languages
- Semantic token highlighting support
